Pemrograman Mobile - Week 01

Pada praktikum Pemrograman Mobile pertemuan pertama, dilakukan instalasi Flutter SDK dengan mengunduh lalu mengekstrak file ke direktori yang mudah diakses di local disk, kemudian menambahkan folder bin ke environment variables Path agar perintah Flutter dan Dart dapat dijalankan melalui terminal. 
Setelah itu instalasi diverifikasi menggunakan perintah flutter --version dan flutter doctor untuk memastikan seluruh dependensi terpenuhi. 
Selanjutnya dilakukan instalasi Android Studio beserta komponen Android SDK hingga berhasil dijalankan dan menampilkan halaman awal sebagai tanda siap digunakan. 
Sebagai uji coba akhir, dibuat program sederhana “Hello World” untuk memastikan konfigurasi Flutter dan Android Studio telah terpasang dengan benar dan dapat digunakan untuk pengembangan aplikasi mobile.
